# MOS edk2

MOS edk2 provides **built and packaged edk2 firmware** for use within the MOS ecosystem.

This repository contains the **build scripts, packaging logic, and configuration files**
required to compile edk2 from upstream sources and package it for MOS.

---

## Overview

edk2 is built **from upstream source code** and packaged for integration into MOS.

No functional changes to edk2 itself are intended.  
The goal of this repository is to provide **consistent, reproducible edk2 packages** for MOS installations.
